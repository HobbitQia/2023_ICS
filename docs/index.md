# 写在前面

你好👋！

这里是浙江大学 2022-2023 短学期《计算机系统概论》[@HobbitQia](https://github.com/HobbitQia) 班的课程主页。

《计算机系统概论》主要内容是从作为开关的晶体管开始，到构建逻辑门，再到更复杂的逻辑结构，然后是带门控的锁存器，最后到存储器的实现。然后学习计算机的指令周期，特定的小教学样机计算机 LC-3 的实现。
